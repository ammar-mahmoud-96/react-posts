# React Blog Website.

React app that displays a list of posts, authors of the posts, number of likes for each post and the comments for each post.

## concepts Are Used :

      1. Reacts Hooks
      2. React Routing
      3. Pagination
      4. Diffrent Sorting
      5. React advanced hooks

## Step to Start Project

- git clone REPO_URL
- cd react-posts
- yarn
- yarn start
